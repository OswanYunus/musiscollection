# 3D Product Landing Page

A minimal Next.js skeleton for a 3D product landing page using Tailwind CSS, Framer Motion, and React Three Fiber.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `pages/index.js`: Main landing page with placeholder for 3D product component.
- `components/3DProduct.js`: Empty React component for Three.js/React Three Fiber integration.
- `styles/globals.css`: Global styles with Tailwind CSS directives.

## Adding 3D Content

- In `components/3DProduct.js`, add your 3D models, lights, animations, and interactions inside the `<Canvas>` component.
- Customize the landing page content in `pages/index.js`.

## Dependencies

- Next.js
- Tailwind CSS
- Framer Motion
- React Three Fiber
- Three.js
